# F1 Data Visualization

This module provides visualization utilities for F1 data analysis.

## Visualization Types

### Race Analysis
- Race progression charts
- Lap time distributions
- Position changes over time
- Gap analysis

### Championship Tracking
- Points progression over season
- Constructor vs driver standings
- Historical comparisons

### Performance Metrics
- Driver performance radar charts
- Constructor comparison plots
- Circuit performance heatmaps

### Statistical Visualizations
- Distribution plots
- Correlation matrices
- Trend analysis charts
- Prediction confidence intervals
