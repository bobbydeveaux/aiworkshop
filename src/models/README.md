# F1 Predictive Models

This module contains machine learning and statistical models for F1 predictions.

## Model Types

### Race Prediction Models
- Race winner prediction based on qualifying
- Points scoring probability
- DNF probability models
- Fastest lap prediction

### Championship Models
- Driver championship probability
- Constructor championship prediction
- End-of-season points prediction

### Performance Models
- Lap time prediction models
- Tire degradation models
- Weather impact models
- Circuit-specific performance models
