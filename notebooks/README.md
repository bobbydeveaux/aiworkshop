# F1 Data Analysis Notebooks

This directory contains Jupyter notebooks for various F1 data analyses.

## Notebook Categories

### Exploratory (`/exploratory/`)
Initial data exploration and understanding:
- API data exploration
- Data quality assessment
- Basic statistical summaries
- Initial visualizations

### Analysis (`/analysis/`)
Detailed analytical notebooks:
- Driver performance analysis
- Constructor comparison studies
- Circuit characteristic analysis
- Historical trend analysis
- Statistical significance testing

### Reports (`/reports/`)
Final analysis reports and findings:
- Season summary reports
- Championship analysis
- Performance insights
- Predictive model results

## Getting Started

1. Install Jupyter: `pip install jupyter`
2. Start Jupyter: `jupyter notebook`
3. Navigate to the appropriate directory
4. Open an existing notebook or create a new one

## Notebook Naming Convention
- `01_data_exploration.ipynb`
- `02_driver_analysis.ipynb`
- `03_constructor_performance.ipynb`
- etc.

Use descriptive names that indicate the analysis focus.
